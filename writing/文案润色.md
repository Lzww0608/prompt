# 📝 写作与内容 - 文案润色

## 基础润色优化

**描述：** 优化文案的语言表达和流畅度

```
你是一名专业编辑。请帮我润色以下文案：

**原文：**
[请粘贴需要润色的文案]

**润色目标：**
[请说明你希望达到的效果，例如：更简洁、更专业、更有感染力等]

**要求：**
1. 保持原文的核心信息和风格
2. 改进语言表达，使其更流畅自然
3. 修正语法和标点错误
4. 优化句子结构和段落组织
5. 使用更准确的词汇
6. 保持原文的字数范围（除非另有要求）

**输出格式：**
请输出：
- 润色后的文案
- 修改说明（列出主要的改动和理由）

请输出润色后的文案。
```

## 专业度提升

**描述：** 提升文案的专业性和权威感

```
你是一名商务写作专家。请帮我提升以下文案的专业度：

**原文：**
[请粘贴需要提升的文案]

**应用场景：**
[例如：商业计划书、工作汇报、产品白皮书]

**目标受众：**
[描述阅读者，例如：投资人、高层领导、客户]

**要求：**
1. 使用更专业、精准的词汇
2. 采用商务写作的语气和风格
3. 增强逻辑性和说服力
4. 使用数据或证据支撑观点
5. 适当使用专业术语（避免过度）
6. 保持简洁明确，避免冗余
7. 格式规范，层次清晰

**输出格式：**
请输出：
- 提升后的专业文案
- 改动说明（解释为什么这样改能提升专业度）

请输出提升后的文案。
```

## 情感化润色

**描述：** 增强文案的情感共鸣和感染力

```
你是一名情感文案专家。请帮我为以下文案注入情感：

**原文：**
[请粘贴基础文案]

**目标情感：**
[请说明希望传达的情感，例如：温暖、激励、紧迫、怀旧等]

**目标受众：**
[描述受众群体的特征和情感需求]

**要求：**
1. 使用有画面感和温度的语言
2. 引发读者的情感共鸣
3. 讲故事的方式传递信息
4. 使用适当的修辞手法（比喻、排比等）
5. 创造沉浸式的阅读体验
6. 结尾有力，留下深刻印象
7. 保持真诚，避免矫揉造作

**输出格式：**
请输出情感润色后的文案。

请输出有感染力的文案。
```

## 简洁化改写

**描述：** 精简文案，用更少的字传达更多信息

```
你是一名简洁写作专家。请帮我精简以下文案：

**原文：**
[请粘贴需要精简的文案]

**目标字数：**
[请说明期望的字数范围]

**使用场景：**
[例如：广告语、标题、社交媒体配文]

**要求：**
1. 删除冗余和重复的信息
2. 用更精炼的表达替代冗长的描述
3. 保留核心信息和关键亮点
4. 使用短句，提高可读性
5. 每个字都有价值，避免废话
6. 保持原文的风格和语气

**输出格式：**
请输出：
- 精简后的文案
- 字数统计
- 删减说明（解释删减的内容和理由）

请输出精简后的文案。
```

## 多版本润色

**描述：** 提供多个润色版本供选择

```
你是一名创意文案专家。请为以下文案提供多个润色版本：

**原文：**
[请粘贴原始文案]

**需要的版本数量：** [例如：3个]

**每个版本的要求：**
[请分别说明每个版本的风格或侧重点]

**要求：**
1. 每个版本保持核心信息一致
2. 每个版本有明显的风格差异
3. 版本 A：[例如：专业正式版]
4. 版本 B：[例如：活泼亲切版]
5. 版本 C：[例如：简洁有力版]
6. 为每个版本说明适用场景

**输出格式：**
请按照以下格式输出：
### 版本 A：[版本标题]
[润色文案]

**适用场景：** [说明适用场景]

### 版本 B：[版本标题]
[润色文案]

**适用场景：** [说明适用场景]

...

请输出多个版本的润色文案。
```

## SEO 优化润色

**描述：** 优化文案以提升搜索引擎排名

```
你是一名 SEO 写作专家。请帮我优化以下文案的 SEO 效果：

**原文：**
[请粘贴需要优化的文案]

**目标关键词：**
[列出主要关键词]

**次要关键词：**
[列出次要关键词或长尾词]

**发布平台：**
[例如：网站博客、微信公众号、知乎]

**要求：**
1. 自然地融入关键词，避免堆砌
2. 优化标题，包含核心关键词
3. 撰写有吸引力的 Meta 描述
4. 使用清晰的标题结构（H1, H2, H3）
5. 优化段落长度，提高可读性
6. 添加相关话题标签
7. 考虑用户搜索意图

**输出格式：**
请输出：
- 优化后的文案（包含所有修改）
- SEO 改动说明
- 关键词使用统计

请输出 SEO 优化后的文案。
```
