# 📝 写作与内容 - 博客撰写

## 技术博客写作

**描述：** 撰写高质量的技术文章

```
你是一名技术作家，擅长将复杂的技术概念用通俗易懂的方式表达出来。请帮我撰写一篇技术博客：

**主题：** [请输入博客主题]

**目标受众：**
[请描述目标读者，例如：初学者、中级开发者、架构师]

**关键词：**
[请列出需要包含的关键词]

**要求：**
1. 标题吸引人且准确
2. 内容结构清晰，有逻辑层次
3. 包含实际的代码示例或实践案例
4. 适当使用图示或比喻解释复杂概念
5. 语言生动有趣，避免过于枯燥
6. 结尾总结要点，提供进一步学习的资源
7. 字数控制在 1500-3000 字之间

**输出格式：**
请使用 Markdown 格式输出，包含以下部分：
- 标题
- 简介（吸引读者）
- 主体（分章节详细讲解）
- 代码示例（如有）
- 总结与展望
- 参考资源

请输出完整的技术博客文章。
```

## 产品测评写作

**描述：** 撰写客观、有深度的产品测评

```
你是一名产品测评专家。请帮我撰写一篇产品测评文章：

**产品名称：** [请输入产品名称]

**产品类型：**
[例如：软件、硬件、SaaS 服务、APP]

**测评重点：**
[请说明你想突出测评的方面，如：性能、易用性、性价比等]

**已知信息：**
[提供产品的基本信息和功能特性]

**要求：**
1. 标题客观中立，带有吸引人的关键词
2. 产品介绍简洁明了
3. 从多个维度进行评测（性能、功能、体验、价格等）
4. 提供真实的使用体验和优缺点分析
5. 对比同类产品的差异
6. 给出适用人群推荐
7. 结论清晰，有说服力

**输出格式：**
请使用 Markdown 格式输出：
- 标题
- 产品简介
- 核心功能
- 详细测评（分维度）
- 优缺点总结
- 适用人群
- 竞品对比
- 购买建议

请输出完整的产品测评文章。
```

## 书评/影评写作

**描述：** 撰写有深度的书籍或电影评论

```
你是一名专业评论家。请帮我撰写一篇评论：

**评论对象：**
[请输入书名或电影名称]

**类型：** [书籍/电影]

**评论角度：**
[例如：文学分析、艺术价值、社会意义、个人感悟]

**要求：**
1. 标题有吸引力
2. 开头介绍作品基本信息和总体评价
3. 从多个角度深入分析（主题、人物、情节、艺术手法等）
4. 避免剧透关键情节（或提前警示）
5. 引用具体内容支撑观点
6. 提供个人见解和思考
7. 结尾给出推荐或避坑建议
8. 字数控制在 1000-2000 字

**输出格式：**
请使用 Markdown 格式输出：
- 标题
- 作品简介
- 详细分析（分章节）
- 亮点与不足
- 个人观点
- 总结与推荐

请输出完整的评论文章。
```
