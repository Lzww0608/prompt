# 📝 写作与内容 - 剧本创作

## 短视频脚本

**描述：** 创作适合抖音、快手等平台的短视频脚本

```
你是一名短视频创作专家。请帮我创作一个短视频脚本：

**视频类型：**
[例如：剧情类、知识科普、搞笑、测评、Vlog]

**主题：** [请输入视频主题]

**目标时长：** [例如：30秒、1分钟、3分钟]

**目标受众：**
[描述目标观众群体]

**核心卖点：**
[说明视频最吸引人的地方]

**要求：**
1. 开头 3 秒抓住注意力（黄金3秒原则）
2. 节奏紧凑，有冲突或反转
3. 对话自然，符合人物性格
4. 包含明确的拍摄指导（画面、动作、运镜）
5. 标注背景音乐（BGM）和音效建议
6. 设计合适的字幕和字幕特效
7. 结尾引导关注、点赞、转发

**输出格式：**
请使用以下格式输出：
| 时间 | 画面描述 | 台词/旁白 | 拍摄指导 | 音效/BGM | 字幕 |
|------|----------|-----------|----------|----------|------|
| 0-3s | [画面] | [内容] | [指导] | [音效] | [字幕] |
| 3-5s | ... | ... | ... | ... | ... |

请输出完整的短视频脚本。
```

## 广告创意脚本

**描述：** 创作有创意和商业价值的广告脚本

```
你是一名广告创意总监。请帮我创作一个广告脚本：

**产品/品牌名称：** [请输入]

**广告类型：**
[例如：电视广告、网络视频广告、电梯广告、地铁广告]

**广告时长：** [例如：15秒、30秒、60秒]

**目标受众：**
[描述目标消费群体]

**核心卖点：**
[说明产品的核心优势或品牌价值]

**广告目的：**
[说明广告要达成的目标，如：品牌认知、促销转化]

**要求：**
1. 创意新颖，有记忆点
2. 开头快速吸引注意力
3. 清晰传达产品/品牌信息
4. 情感共鸣或痛点直击
5. 品牌露出自然
6. 行动呼吁（Call to Action）明确
7. 考虑制作可行性

**输出格式：**
请使用以下格式输出：
**广告标题：** [标题]

**创意概念：** [用一句话概括核心创意]

**脚本正文：**
| 时间 | 画面 | 音效/音乐 | 字幕/旁白 | 备注 |
|------|------|-----------|-----------|------|
| 0-3s | [画面] | [音效] | [内容] | [说明] |
| 3-5s | ... | ... | ... | ... |

**创意说明：** [解释创意的逻辑和预期效果]

请输出完整的广告脚本。
```

## 电影/电视剧剧本

**描述：** 创作专业的影视剧本

```
你是一名专业编剧。请帮我创作一个影视剧本：

**剧本类型：**
[例如：剧情片、喜剧、悬疑片、科幻片、爱情片]

**故事主题：** [请输入核心主题]

**角色设定：**
1. [主角姓名]：[年龄、性格、背景]
2. [配角姓名]：[角色特征]

**场景设定：**
[描述故事发生的时间和地点]

**情节要求：**
[说明具体要写的情节内容]

**要求：**
1. 遵循专业剧本格式
2. 对话自然，符合人物性格
3. 动作描写生动
4. 节奏张弛有度
5. 包含场景转换和时间指示
6. 注明镜头运动（如适用）
7. 有戏剧冲突和情感张力

**输出格式：**
请使用标准剧本格式输出：
```
场景：[地点] - [时间]

[人物]：
[对话内容]

[动作/场景描述]
...
```

请输出完整的剧本片段。
```

## 直播脚本

**描述：** 创作直播带货或娱乐直播的脚本

```
你是一名直播运营专家。请帮我创作一个直播脚本：

**直播类型：**
[例如：带货直播、知识直播、游戏直播、秀场直播]

**直播主题：** [请输入]

**直播时长：** [例如：2小时、4小时]

**核心产品/内容：**
[列出主要要推广的产品或内容]

**目标受众：**
[描述直播间观众特征]

**要求：**
1. 设计完整的时间流程
2. 开场暖场，留住观众
3. 产品/内容介绍节奏合理
4. 设计互动环节（抽奖、问答、连麦）
5. 设计促单话术和紧迫感营造
6. 预留突发情况应对方案
7. 结尾引导关注和下次直播

**输出格式：**
请使用以下格式输出：
### 开场阶段（0-15分钟）
- 内容：[描述]
- 话术：[示例话术]
- 互动：[互动设计]

### 产品介绍阶段（15-90分钟）
- 产品 A：[名称、卖点、价格]
- 话术：[推销话术]
- 互动：[提问、抽奖]

### ...其他阶段

### 突发情况应对
- [问题]：[应对方案]

请输出完整的直播脚本。
```

## 播客/音频节目脚本

**描述：** 创作音频播客节目的脚本

```
你是一名播客制作人。请帮我创作一个播客脚本：

**节目名称：** [请输入]

**节目类型：**
[例如：访谈类、知识分享类、故事类、新闻评论类]

**本期主题：** [请输入]

**节目时长：** [例如：30分钟、1小时]

**主持人/嘉宾：**
[说明主持人风格或嘉宾背景]

**要求：**
1. 开头有吸引力，快速进入主题
2. 内容有逻辑层次，层次清晰
3. 对话自然流畅
4. 设计转场和过渡语
5. 包含背景音乐（BGM）和音效提示
6. 考虑听众的收听体验
7. 结尾总结，引导订阅

**输出格式：**
请使用以下格式输出：
### 开场（0-3分钟）
- 主持人：[开场白]
- BGM：[音乐风格]

### 第一部分：[标题]（3-15分钟）
- [内容大纲]
- 主持人/嘉宾对话
- 转场语

### 第二部分：[标题]（15-25分钟）
- ...

### 结尾（25-30分钟）
- 总结
- 订阅引导
- 下期预告

请输出完整的播客脚本。
```

## 互动剧情脚本

**描述：** 创作分支剧情的互动式脚本

```
你是一名互动剧情设计师。请帮我创作一个互动剧情脚本：

**故事背景：**
[描述故事的世界观和背景]

**主角设定：**
[描述主角的身份和特征]

**故事主题：** [请输入核心主题]

**互动设计要求：**
[说明如何设计分支和选择]

**要求：**
1. 主线故事清晰完整
2. 设计多个关键分支选择点
3. 每个选择有合理的后果
4. 分支最终能汇合到不同结局
5. 设计角色好感度或属性系统（如需要）
6. 考虑玩家的沉浸感和代入感
7. 对话选项有辨识度

**输出格式：**
请使用以下格式输出：
### 场景 1：[场景名称]
**背景描述：** [环境描述]

**NPC 对话：**
[角色]："[对话内容]"

**分支选择：**
A. [选项 A] → [后果描述] → 跳转到场景 X
B. [选项 B] → [后果描述] → 跳转到场景 Y

### 场景 2：...
[继续设计后续场景]

### 结局设计：
- 结局 A：[触发条件]
- 结局 B：[触发条件]

请输出完整的互动剧情脚本。
```
