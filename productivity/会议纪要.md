# 📊 职场与生产力 - 会议纪要

## 会议纪要撰写

**描述：** 快速、准确地记录会议要点

```
你是一名会议记录专家。请帮我整理以下会议纪要：

**会议类型：**
[例如：项目会议、头脑风暴、周会、决策会议]

**会议主题：** [会议主题]

**参会人员：**
[列出参会人员姓名和职位]

**会议记录/录音：**
[请粘贴会议记录或关键信息]

**会议时间：** [日期和时间]

**要求：**
1. 结构清晰，层次分明
2. 重点突出，不遗漏关键信息
3. 语言简洁，便于阅读
4. 准确传达决策和行动项
5. 明确责任人和截止日期
6. 避免个人主观判断
7. 使用规范的格式

**输出格式：**
请使用以下格式输出：
# [会议名称] 会议纪要

**会议时间：** [时间]

**参会人员：** [名单]

**记录人：** [姓名]

---

## 一、会议目的
[简要说明会议目标]

## 二、会议内容
### 2.1 [主题 1]
- 讨论要点
- 主要观点
- 决策结果

### 2.2 [主题 2]
...

## 三、会议决策
- 决策 1：[内容]
- 决策 2：[内容]

## 四、行动项
| 序号 | 行动内容 | 负责人 | 截止日期 | 状态 |
|------|----------|--------|----------|------|
| 1 | [内容] | [姓名] | [日期] | 待开始 |
| 2 | [内容] | [姓名] | [日期] | 待开始 |

## 五、下次会议
- 时间：[日期]
- 议题：[待讨论内容]

请输出完整的会议纪要。
```

## 会议总结提炼

**描述：** 从冗长的会议内容中提炼核心信息

```
你是一名信息提炼专家。请帮我从以下会议记录中提炼核心信息：

**会议记录：**
[请粘贴会议记录内容]

**关注重点：**
[说明你最关注的信息，如：决策、行动项、风险等]

**要求：**
1. 提取核心信息和决策
2. 识别关键行动项
3. 总结会议成果和意义
4. 识别风险和挑战
5. 概括各方观点
6. 语言精炼，直击要点
7. 提供后续跟进建议

**输出格式：**
请使用以下格式输出：
# 会议核心要点

## 一、关键决策
1. [决策 1]
2. [决策 2]

## 二、重要行动项
| 行动内容 | 负责人 | 截止时间 |
|----------|--------|----------|
| [内容] | [姓名] | [时间] |
| [内容] | [姓名] | [时间] |

## 三、会议成果
- [成果 1]
- [成果 2]

## 四、风险与挑战
- [风险 1]
- [风险 2]

## 五、跟进建议
- [建议 1]
- [建议 2]

请输出提炼后的会议总结。
```

## 会议议程设计

**描述：** 设计高效、有条理的会议议程

```
你是一名会议策划专家。请帮我设计以下会议的议程：

**会议主题：** [请输入]

**会议类型：**
[例如：项目启动会、需求评审会、复盘会、头脑风暴]

**会议时长：** [例如：30分钟、1小时、2小时]

**参会人员：**
[描述参会人员的角色和背景]

**会议目标：**
[请说明会议要达成的具体目标]

**已知议题：**
[列出已经确定需要讨论的议题]

**要求：**
1. 议程安排合理，符合会议目标
2. 时间分配恰当，避免拖沓
3. 逻辑顺序清晰，便于讨论
4. 预留缓冲时间
5. 明确每个环节的负责人和输出
6. 设计互动环节
7. 提供议程执行说明

**输出格式：**
请使用以下格式输出：
# [会议名称] 议程

**会议时间：** [日期和时间]

**会议时长：** [总时长]

**会议地点：** [地点]

**参会人员：** [名单]

**会议目标：** [简要说明]

---

| 时间 | 议题 | 负责人 | 预期输出 | 说明 |
|------|------|--------|----------|------|
| 0-5min | 开场和议程确认 | [姓名] | 确认目标 | 说明会议规则 |
| 5-15min | 议题 1 | [姓名] | 决策/方案 | 讨论要点 |
| 15-25min | 议题 2 | [姓名] | 行动项 | [说明] |
| 25-30min | 总结和下一步 | [姓名] | 行动计划 | 明确分工 |

**议程说明：**
- [说明 1]
- [说明 2]

请输出完整的会议议程。
```

## 会议准备清单

**描述：** 为不同类型的会议创建准备清单

```
你是一名会议准备专家。请帮我创建会议准备清单：

**会议类型：**
[例如：客户演示会、项目评审会、面试会、跨部门协调会]

**会议主题：** [请输入]

**你的角色：**
[你在会议中的角色，如：主持人、参会者、记录员]

**会议重要性：**
[说明会议的重要程度]

**要求：**
1. 列出会前需要准备的事项
2. 区分必须准备和可选准备
3. 包含材料、设备、人员等方面
4. 提供时间规划建议
5. 设计检查清单格式
6. 提供应急准备方案

**输出格式：**
请使用以下格式输出：
# [会议名称] 准备清单

## 一、必准备项（提前 1-2 天）
### 1.1 材料准备
- [ ] [材料 1]
- [ ] [材料 2]

### 1.2 人员协调
- [ ] [确认参会人员]
- [ ] [邀请嘉宾]

### 1.3 场地与设备
- [ ] [会议室预订]
- [ ] [设备调试]

## 二、会前检查（会议前 30 分钟）
- [ ] [检查项 1]
- [ ] [检查项 2]

## 三、可选准备
- [ ] [可选 1]
- [ ] [可选 2]

## 四、应急方案
- [应急情况 1]：[应对措施]
- [应急情况 2]：[应对措施]

请输出完整的会议准备清单。
```

## 会议效果评估

**描述：** 评估会议的有效性和改进建议

```
你是一名会议管理专家。请帮我评估以下会议的效果：

**会议基本信息：**
- 会议主题：[主题]
- 参会人数：[人数]
- 会议时长：[时长]
- 会议类型：[类型]

**会议目标：** [会议目标]

**实际成果：**
[说明会议达成的成果]

**存在的问题：**
[描述会议中存在的问题或不足]

**参会反馈：**
[如有参会者的反馈，请说明]

**要求：**
1. 评估会议是否达成目标
2. 分析会议的优点和不足
3. 识别效率问题和瓶颈
4. 提供改进建议
5. 设计会议效果指标
6. 提供下次会议优化方案

**输出格式：**
请使用以下格式输出：
# 会议效果评估报告

## 一、目标达成情况
- 目标 1：[达成/未达成] - [说明]
- 目标 2：[达成/未达成] - [说明]

## 二、会议优点
- 优点 1：[具体说明]
- 优点 2：[具体说明]

## 三、存在的问题
- 问题 1：[描述] - [影响程度]
- 问题 2：[描述] - [影响程度]

## 四、改进建议
1. [建议 1]
2. [建议 2]

## 五、效率指标
- 决策效率：[评分 1-10] - [说明]
- 参与度：[评分 1-10] - [说明]
- 时间控制：[评分 1-10] - [说明]

## 六、下次会议优化方案
- [优化 1]
- [优化 2]

请输出完整的会议效果评估报告。
```
