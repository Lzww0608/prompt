# 🎓 学术与教育 - 论文润色

## 语言润色

**描述：** 提升学术论文的语言表达质量

```
你是一名学术英语编辑，专门帮助研究者润色学术论文。请帮我润色以下文本：

**文本类型：**
[例如：摘要、引言、方法、结果、讨论、结论]

**原文：**
[请粘贴需要润色的文本]

**润色要求：**
1. 提升语言的学术性和正式性
2. 改善句子结构，使表达更清晰
3. 使用更精准的学术词汇
4. 修正语法错误和不当表达
5. 保持原文的核心观点和数据
6. 符合学术写作规范
7. 中英文对照（如果需要）

**输出格式：**
请输出：
- 润色后的文本
- 修改说明（列出主要改动及理由）
- 如适用，提供多个版本供选择

请输出润色后的文本。
```

## 逻辑结构优化

**描述：** 优化论文的逻辑框架和论证结构

```
你是一名学术写作顾问。请帮我优化论文的逻辑结构：

**论文主题：** [请输入]

**当前结构：**
1. [章节 1]
2. [章节 2]
3. [章节 3]
...

**论文内容摘要：**
[请简要描述论文的核心内容和研究问题]

**存在的问题：**
[说明你认为论文结构上存在的具体问题]

**要求：**
1. 分析当前结构的优缺点
2. 提出改进的章节安排
3. 优化论证的逻辑链条
4. 确保研究问题、方法、结果、结论的逻辑一致性
5. 考虑读者的阅读体验
6. 提供详细的改写建议

**输出格式：**
请输出：
- 当前结构分析
- 优化后的章节大纲
- 每个章节的核心内容建议
- 逻辑关系说明

请输出优化后的论文结构方案。
```

## 摘要写作

**描述：** 撰写高质量的学术论文摘要

```
你是一名学术写作专家。请帮我撰写论文摘要：

**论文信息：**
- 标题：[论文标题]
- 研究领域：[所属学科]
- 研究方法：[例如：实验研究、文献综述、案例研究]

**研究内容：**
[请简要描述研究的主要内容和发现]

**关键发现：**
[列出 2-3 个最重要的研究发现]

**研究意义：**
[说明研究的创新点和贡献]

**要求：**
1. 清晰阐述研究背景和目的
2. 简要描述研究方法
3. 突出主要发现和结论
4. 说明研究的意义和影响
5. 字数控制在 150-250 词（英文）或 300-500 字（中文）
6. 语言简洁有力
7. 包含关键词

**输出格式：**
请使用以下格式输出：
**摘要**

[摘要正文]

**关键词：** [关键词1, 关键词2, 关键词3]

请输出完整的摘要。
```

## 引言写作

**描述：** 撰写有说服力的论文引言

```
你是一名学术写作专家。请帮我撰写论文引言：

**论文标题：** [请输入]

**研究领域：** [所属学科]

**研究问题：**
[明确陈述论文要解决的核心问题]

**研究背景：**
[请提供研究背景信息]

**要求：**
1. 引入研究主题，说明重要性
2. 综述相关文献（研究现状）
3. 指出研究空白或不足
4. 明确本研究的创新点和贡献
5. 概述研究方法和预期结果
6. 逻辑清晰，层层递进
7. 引用相关文献（用 [作者, 年份] 格式）

**输出格式：**
请使用 Markdown 格式输出：
- 第一段：研究背景和重要性
- 第二段：文献综述和现状
- 第三段：研究空白和不足
- 第四段：本研究的目标和贡献

请输出完整的论文引言。
```

## 结论写作

**描述：** 撰写有力的论文结论部分

```
你是一名学术写作专家。请帮我撰写论文结论：

**论文标题：** [请输入]

**主要发现：**
[列出研究的关键发现和结果]

**研究目标：**
[回顾论文最初的研究目标]

**要求：**
1. 总结主要发现，回答研究问题
2. 强调研究的创新点和贡献
3. 讨论研究的局限性
4. 提出未来研究方向
5. 避免引入新的结果或数据
6. 语言简洁，避免重复
7. 提升文章的学术价值

**输出格式：**
请使用以下格式输出：
**结论**

[结论正文，建议 3-4 段]

### 研究局限性

[说明研究的不足和限制]

### 未来研究方向

[提出值得进一步研究的方向]

请输出完整的论文结论。
```

## 引文格式规范

**描述：** 规范化论文的引用格式

```
你是一名学术写作专家。请帮我规范引文格式：

**引文格式：**
[选择格式，例如：APA、MLA、Chicago、IEEE]

**原文内容：**
[请粘贴包含引文的文本]

**参考文献信息：**
[列出引用的文献基本信息]

**要求：**
1. 根据选定的引文格式规范化所有引用
2. 检查文内引用（In-text Citation）格式
3. 规范化参考文献列表格式
4. 确保文内引用与参考文献列表一致
5. 标注需要补充信息的引用
6. 提供格式示例说明

**输出格式：**
请输出：
- 规范化后的文本（包含正确的文内引用）
- 规范化后的参考文献列表
- 修改说明

请输出规范化后的引文内容。
```
