# 💻 编程与开发 - 代码生成

## 基础代码生成

**描述：** 根据需求描述生成功能代码，适用于快速原型开发

```
你是一名拥有 10 年经验的资深 [编程语言] 工程师。请根据以下需求编写代码：

**需求描述：**
[请输入你的功能需求]

**要求：**
1. 代码需要遵循最佳实践和设计模式
2. 添加必要的注释说明关键逻辑
3. 包含错误处理机制
4. 代码结构清晰，易于维护
5. 提供一个简单的使用示例

**输出格式：**
请使用 Markdown 代码块输出，并按照以下结构：
```[编程语言]
// 完整代码实现
```

**使用示例：**
```[编程语言]
// 示例代码
```
```

**描述：** 根据算法描述生成实现代码

```
你是一名算法专家。请帮我实现以下算法：

**算法名称：** [请输入算法名称]

**算法描述：**
[请详细描述算法的输入、输出和核心逻辑]

**输入：**
- [参数1]
- [参数2]

**输出：**
- [返回值描述]

**要求：**
1. 时间复杂度和空间复杂度尽可能优化
2. 添加详细注释说明算法步骤
3. 提供时间复杂度和空间复杂度分析
4. 给出 2-3 个测试用例

请输出完整的代码实现和复杂度分析。
```

## API 接口设计

**描述：** 设计 RESTful API 接口

```
你是一名后端架构师。请帮我设计以下 API 接口：

**功能描述：**
[请描述 API 需要实现的功能]

**技术栈：**
- 后端框架：[例如：Spring Boot, Express.js, Django]
- 数据库：[例如：MySQL, PostgreSQL, MongoDB]

**要求：**
1. 设计符合 RESTful 风格的 API 端点
2. 定义请求和响应的数据结构（JSON Schema）
3. 包含必要的 HTTP 状态码
4. 考虑身份认证和权限控制
5. 提供错误处理机制

**输出格式：**
请使用 Markdown 表格格式输出 API 文档：
| 方法 | 端点 | 描述 | 请求参数 | 响应 |
|------|------|------|----------|------|
```

## 单元测试编写

**描述：** 为现有代码生成单元测试

```
你是一名测试工程师。请为以下代码编写单元测试：

**代码实现：**
```[编程语言]
[请粘贴需要测试的代码]
```

**要求：**
1. 使用主流的测试框架（如 JUnit、pytest、Jest）
2. 覆盖正常情况、边界情况和异常情况
3. 使用 Mock/Stub 处理外部依赖
4. 测试用例命名清晰，易于理解
5. 目标代码覆盖率至少 80%

**输出格式：**
请提供完整的测试代码，并简要说明每个测试用例的目的。
```

## 代码重构

**描述：** 对现有代码进行优化重构

```
你是一名代码审查专家。请对以下代码进行重构优化：

**原始代码：**
```[编程语言]
[请粘贴需要重构的代码]
```

**问题说明：**
[可以描述当前代码存在的问题，如：可读性差、性能问题、违反设计原则等]

**重构目标：**
[请说明你希望达到的优化效果]

**要求：**
1. 提高代码可读性和可维护性
2. 遵循 SOLID 原则和设计模式
3. 优化性能（如适用）
4. 保持原有功能不变
5. 对比重构前后的差异

**输出格式：**
请输出重构后的代码，并说明：
- 重构的具体改动
- 改动带来的好处
- 可能存在的风险
```
